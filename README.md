# C-Projects

This repository contains a collection of C++ console-based programs created to practice and demonstrate fundamental programming concepts. Each file represents an independent project focused on solving a specific problem using C++.

## Contents

- BankAccountManagementSystem.cpp  
  Basic bank account operations such as creating accounts, depositing, withdrawing, and checking balances.

- CommandLineAuthentication.cpp  
  Simple command-line user authentication system.

- GameProject.cpp  
  Console-based game demonstrating control flow and user interaction.

- Gym.cpp  
  Gym management system for handling member information.

- LibraryManagementSystem.cpp  
  Library system for managing books and issuing/returning records.

- Search_engine.cpp  
  Basic text-based search functionality.

- StudentManagementSystem.cpp  
  Student record management system.

- StudentMngSyst.cpp  
  Alternative implementation of a student management system.

- TodoListApp.cpp  
  Simple to-do list application for task management.

- main.cpp  
  Entry point for testing or running programs.

## Language

- C++ (100%)
